## 三、技术架构要求

### 前端
- Vue.js 2.x (兼容TRAINSET组件)
- Chart.js / D3.js 图表库
- 响应式三栏布局

### 后端
- Flask REST API
- Pandas 数据处理
- JSON 标注存储

### 集成目标
- 使用TRAINSET交互模式（拖拽选区、点击标注）
- 使用V1后端API（路径管理、标注存储）
- 输出符合ChatTS训练格式的JSON
